# Image Editor
A small image editor site whre you can change brightness, contrast, saturation, hue, grayscale, sepia, opacity, blur and invert. If you dont know how to edit images, you can use the presets available.

## Features
- Responsive for all devices
- Everything is client side so no fear of image being stolen
- Nice and easily understandable UI
- Editing presets
- If you dont like the edit you did, you can reset everything
- Edited image downloadable

## Languages Used
- HTML
- CSS
- Javascript

## Instructions to see the site
### On your device
- Open github
- search rojen-chakradhar
- go to repo
- look for image-editor
- clone the repo
- run index..html file
- the site is open
- now you can edit images

### Deployed
- Open any browser you want (chrome for best experience)
- search [rojens-tools.netlify.app](https://rojens-tools.netlify.app)
- then the site is open
- now you can edit images
